Discover and document a TopTracker API endpoint by calling it.

Instructions: $ARGUMENTS

You have access to the TopTracker API at https://tracker-api.toptal.com. To discover an endpoint:

1. First authenticate by calling POST /sessions with credentials (ask the user for credentials if not available)
2. Call the endpoint the user described using curl
3. Document the full request (method, path, query params, body) and response shape
4. Suggest types and where they should be added in the SDK
